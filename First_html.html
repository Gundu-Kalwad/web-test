HTML Basics: Questions and Answers

1. What is HTML? Explain features of HTML.

HTML (HyperText Markup Language) is the standard markup language used to create webpages. It describes the structure of a webpage using a set of elements and tags. Web browsers read HTML documents and display them as webpages.

Features of HTML:

  1. Simple and Easy – HTML is easy to learn and implement.
  2. Platform Independent – Works on any operating system and browser.
  3. Hyperlink Support – Allows linking between pages using <a> tag.
  4. Multimedia Support – Can embed images, audio, video, and animations.
  5. Structured Content – Provides headings, lists, tables, forms, etc.
  6. Supported Everywhere – All modern browsers support HTML.

2. What is HTML document?

An HTML document is a file with .html or .htm extension that contains HTML code. It is the main unit of a webpage, read and interpreted by a web browser.

Main parts of an HTML document:

  1. DOCTYPE Declaration → Defines document type (<!DOCTYPE html>).
  2. Root Element <html> → Contains all other elements.
  3. Head Section <head> → Metadata, title, CSS, JavaScript.
  4. Body Section <body> → Visible content such as text, images, links.

So, an HTML document is the backbone of a webpage that organizes and displays information.

3. Explain the basic structure of an HTML document.

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Welcome</h1>
    <p>This is a sample webpage.</p>
  </body>
</html>

Explanation:

  • <!DOCTYPE html> – Defines HTML version.
  • <html> – Root of the page.
  • <head> – Contains title, metadata, links to CSS/JS.
  • <title> – Name shown in browser tab.
  • <body> – Main content displayed to users.

This structure is mandatory for every HTML document.

4. Explain the following tags with an example.

a) Header tag <h1> to <h6>

Used for headings. <h1> is the largest, <h6> is the smallest.

<h1>Main Title</h1>
<h3>Sub Title</h3>

b) Paragraph tag <p>

Defines a block of text.

<p>This is a paragraph.</p>

c) Subscript <sub> and Superscript <sup>

Used in scientific and mathematical expressions.

H<sub>2</sub>O → Water formula
x<sup>2</sup> + y<sup>2</sup> → Equation

5. Explain an HTML element where we demonstrate working with text tags.

HTML provides formatting tags to style text. For example:

<p><b>Bold</b>, <i>Italic</i>, <u>Underline</u>, <mark>Highlighted</mark></p>

Explanation:

  • <b> → Bold text.
  • <i> → Italic text.
  • <u> → Underlined text.
  • <mark> → Highlights text.

Thus, text tags allow emphasis and formatting to improve readability of web content.

6. Explain table tag along with its specific property.

Tables are created using <table> element in HTML.

<table border="1" cellpadding="5" cellspacing="2">
  <tr><th>Name</th><th>Age</th></tr>
  <tr><td>John</td><td>20</td></tr>
  <tr><td>Ana</td><td>22</td></tr>
</table>

Properties/Attributes:

  1. border – Defines table border.
  2. cellpadding – Space inside a cell.
  3. cellspacing – Space between cells.
  4. align – Alignment of table.
  5. width – Sets table width.

7. What is a list? Explain different types of lists.

A list is a collection of related items presented in an organized way. HTML supports three types:

1. Ordered List <ol> – Items are numbered.

<ol><li>Apple</li><li>Banana</li></ol>

2. Unordered List <ul> – Items are bulleted.

<ul><li>Milk</li><li>Bread</li></ul>

3. Definition List <dl> – Contains terms and definitions.

<dl>
  <dt>HTML</dt><dd>Markup Language</dd>
  <dt>CSS</dt><dd>Styling Language</dd>
</dl>

8. Explain different form controls and elements with an example.

Forms are used to collect user input. Common controls are:

  • Textbox: <input type="text">
  • Password field: <input type="password">
  • Radio button: <input type="radio" name="g"> Male
  • Checkbox: <input type="checkbox"> Music
  • Textarea: <textarea></textarea>
  • Dropdown: <select><option>Option1</option></select>
  • Button: <input type="submit" value="Submit">

Example:

<form>
  Name: <input type="text"><br>
  Gender: <input type="radio" name="g"> Male
          <input type="radio" name="g"> Female <br>
  <input type="submit" value="Submit">
</form>

9. Write a short note on:

a) Working with Hyperlinks

Created using <a> tag with href attribute.

Used to link web pages, sections, or external sites.

<a href="https://example.com">Visit Example</a>

Can also link to email (mailto:) or file downloads.

b) Frames

Used to divide the browser window into multiple sections, each showing a different document.

<frameset cols="50%,50%">
  <frame src="left.html">
  <frame src="right.html">
</frameset>

Now replaced by CSS and <iframe> for better practice.

c) Images and Multimedia

Image: <img src="pic.jpg" alt="Description" width="200">

Audio:

<audio controls>
  <source src="song.mp3" type="audio/mpeg">
</audio>

Video:

<video controls width="300">
  <source src="movie.mp4" type="video/mp4">
</video>

These tags enhance user experience with visual and audio elements.
